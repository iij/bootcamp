## 概要

[IIJ](https://www.iij.ad.jp/) では　「IIJ Bootcamp」　と題して主に新人向けのハンズオン勉強会を実施しており、これはその資料集です。
ハンズオンは「実際に触ってみる」ことを目的として、各分野で実際に業務を行なっている技術者が1h~2h枠で実施できる形にまとめています。
